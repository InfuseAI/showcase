# Use sqlpad in PrimeHub apps

## Prerequisites

- PrimeHub
    
## Usage

- Method 1:
```bash
kubectl -n hub apply -f sqlpad.yaml
```

- Copy link and import yaml file in PrimeHub admin UI.
